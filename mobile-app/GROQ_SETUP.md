## 🚀 Setup (2 Minutes)

### Step 1: Get Groq API Key

1. Visit: **https://console.groq.com/**
2. Sign up (Google/GitHub/Email - it's free!)
3. Go to: **https://console.groq.com/keys**
4. Click **"Create API Key"**
5. Give it a name (e.g., "Black Pepper Chatbot")
6. Copy the key (starts with `gsk_...`)

**NO credit card needed!** 🎉

---

### Step 2: Add API Key to Your App

**Open:** `lib\services\gemini_service.dart`

**Line 7 - Replace:**
```dart
static const String _apiKey = 'YOUR_GROQ_API_KEY_HERE';
```

**With your actual key:**
```dart
static const String _apiKey = 'gsk_your_actual_groq_api_key_here';
```

---

### Step 3: Hot Restart

Press **Shift + R** in your terminal (where Flutter is running)

### Step 4: Test the Chatbot!

1. Open app → Farmer Dashboard
2. Tap "Ask AI Assistant" card
3. Ask: **"What is yellow mottle virus?"**
