# ML Based Integrated Solution for Smart Pepper Farming in Sri Lanka

This project is the Final Year Research Project conducted at the Sri Lanka Institute of Information Technology (SLIIT). The study proposes an integrated smart agriculture system for the Sri Lankan pepper industry. It addresses key challenges faced by small-scale farmers such as unfair pricing, poor quality grading, late disease detection, and uncertain yields.

By combining machine learning, mobile technology, and blockchain, the system provides data-driven insights, transparency, and decision support for farmers, exporters, and regulators.

## 🚜 Problem Statement

The Sri Lankan pepper sector faces several critical issues:

- Unpredictable pricing causes income loss for farmers.<br>
- Traditional quality grading relies only on 1L cup density, ignoring mold and adulteration.<br>
- Late disease detection leads to irreversible crop damage.<br>
- Uncertain yield estimation results in poor planning and wasted resources.<br>

These issues negatively impact farmers’ livelihoods and weaken Ceylon pepper’s global reputation.

## 💡 Proposed Solution

Our system provides a smart solution with four main features:

- **Yield Prediction:**
Uses machine learning to estimate crop yield and help farmers plan resources and income.

- **Price Forecasting with Blockchain:**
Predicts future pepper prices and uses blockchain to ensure fair and transparent transactions.

- **Quality Grading:**
Uses images to check pepper quality by detecting mold, adulteration, and size differences.

- **Disease Detection:**
Identifies plant diseases early and shows how much the infection has spread for timely action.

## 👥 Target Users

- Primary: Small-scale pepper farmers in Sri Lanka<br>
- Secondary: Exporters, buyers, agricultural officers, and regulators<br>

**Flutter**<br>

- Farmer-friendly UI for predictions, alerts, and insights

### 🌐 Backend

**Node.js**<br>

### 🤖 Machine Learning

**Python**<br>

- Yield prediction<br>
- Price forecasting<br>
- Disease detection<br>
- Quality grading

### ⚡ API Layer

**FastAPI**<br>
- Serves ML model predictions<br>
- High-performance REST APIs for real-time inference

## 🎯 Key Contributions

- Integrated forecasting, quality analysis, disease detection, and traceability in a single system<br>
- Improves farmers’ decision-making and income planning<br>
- Enhances export quality standards and transparency<br>
- Demonstrates real-world application of ML and blockchain in agriculture<br>
